# Tienda de Componentes

Proyecto académico que simula una tienda online de componentes de ordenador, desarrollado en HTML, CSS y JavaScript. Este sitio web permite visualizar productos como tarjetas gráficas y procesadores, y añadirlos al carrito mediante interacción básica.

## Estructura del Proyecto
├── index.html --> Pagina principal

├── styles.css --> Estilos

├── scripts.js --> Script para añadir un producto al carrito y el envio del formulario

└── img --> Imagenes usadas en el codigo

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Diseño responsivo y estilización
- **JavaScript**: Función `agregarCarrito()` para simular la acción de compra

## Conclusión Académica
Este proyecto demuestra la aplicación práctica de HTML, CSS y JavaScript en la creación de una tienda 
web funcional. Se ha trabajado la estructura semántica, el diseño responsivo y la interacción básica 
con el usuario. Además, se identificaron y corrigieron errores visuales, reforzando la importancia de 
la revisión técnica.

Desde una perspectiva ética, se invita a reflexionar sobre el impacto ambiental del hardware y la 
necesidad de fomentar el consumo responsable. El ejercicio fortalece competencias técnicas y críticas 
en el desarrollo web académico.
